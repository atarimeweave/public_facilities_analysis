\# 行政オープンデータの活用例として、分析利用可能性の探索

\## 概要

昨今データ活用目的のため、各自治体でオープンデータが公開されている。自治体でのデータ活用は主に情報マッピングであり、HPにて地図連携で各情報が公開されている事例もある。

では、オープンデータを用いて"データ分析"はどこまでできるのか。

今回は、

* 地域活動の拠点として「公民館」
* 居住環境として「市営住宅」
* 地域の規模として「地域の人口」

この3つの情報と地域構造との関係性を観察した。

\---

\## 背景

大規模都市ではデータが多い分、オープンデータも豊富であることは予測可能であるが、地方都市でも専用HPにてオープンデータが整理されて公開されていることが確認できた。

HPのオープンデータいの中身をいくつか見てみると、多くが施設一覧や管理台帳に近いものであり、数値データは位置情報のみ、という状態。

位置情報はマッピングのための情報と推測する。

他の記載情報を使用しマッピング以外の活用方法、例えば

「"分析"は情報の組み合わせにより可能ではないか」

と仮説を立てた。

\---

\## 分析目的

* "オープンデータのみ"での地域分析の可能性を確認する
* オープンデータはないが、HP等で紹介されている公開情報を追加した場合、どのように分析可能性が広がるか検証する

\---

\## 使用データ

公共施設データ		公民館の住所と位置情報情報の抽出

市営住宅データ		市営住宅の住所と位置情報

人口統計データ　	年齢別、地域別の人口

独自収集データ		公民館の貸出可能部屋数

\---		

\## 分析手順

1. データ確認 (SQL)
-  カラム確認		必要カラム抽出
- カラム名変更		英語表記に変更
- 地域名の整備		住所から地域名のみを抽出　
- 距離計算			公民館と各市営住宅の距離を算出し、
一番近い市営住宅と紐づけ
- 地域名を丸める	「丁目」などの地域を地域名でまとめ、
人口を集計
2. データ結合 (SQL)
地域名をキーとして公民館（市営住宅と紐づけ済み）と人口データを結合
3. 分析 (Python:jupyter notebook)
- 公民館を軸に集計、可視化
- 地域を軸に集計、可視化、クラスター分析
4. 属性情報を追加して分析 (Python:jupyter notebook)

&#x09;- 各公民館のHPより貸出可能な部屋数を収集、施設規模の指標として追加

&#x20;   - 再び公民館を軸に、貸出可能部屋数と各情報との関係性を確認

\---

\## 仮説と検証の流れ



\### 仮説1

公民館は「地域活動の場」として人口の多い地域にあるのではないか。

また、人口の多い地域は住宅街を含むため、結果的に公民館は市営住宅の近くに配置されるのではないか。

\### 検証

公民館と最寄りの市営住宅との距離を算出。

公民館がある地域の人口と人口比を算出。

これらについて比較した。

\### 結果

* 外れ値を除くと多くの公民館は最寄り市営住宅から約2km以内に位置している
* 自分の知識として人口が多そうな地域の市営住宅から最寄り公民館までは3km離れていた
* 最寄り市営住宅との距離が遠い公民館は総人口が極端に少ない地域もある
* 最寄り市営住宅との距離がほぼ同じでも地域の人口が約16倍も差がある公民館もある

市営住宅との他の公共施設の距離を比較していないため、約2kmという距離感は「近い」とまでは言い切れない。

また、人口差も多く、公民館がある地域には人口も多い、とも言いきれない。



\### 仮説2

公民館がある地域はない地域に比べて若者人口の割合と総人口が多い。

公民館がある地域の人口構成には特徴がある。

\### 検証

「男女」「年齢3区分（-14,15-64,65-)」の人口比を算出。

特徴量を「①15歳未満人口割合」「②65歳以上人口割合」「③男性人口割合」「④総人口」「⑤公民館フラグ」の5つとし、様々な組み合わせでクラスタリングモデルを比較した。

\### 結果

* クラスタの平均値を比較した結果、③の男性人口割合はほぼ同じ数値だった
* ①②④⑤の特徴量で5つにクラスター分析をした場合、人口に関する情報が多いためか一般的によく見られる人口比構成の特性で分類され、「公民館のある地域」は独立クラスタに分けられた
* 特徴量「公民館フラグ」の有無でクラスター分類結果を見ると、公民館がある地域は人口規模が中～大に分布

子どもの割合が多い地域に公民館があるとは言い切れないが、高齢化地域や人口が特に多い中心地域には公民館がない傾向がある。



\### 仮説3

「市営住宅が近くにある」「人口が多い地域にある」ほど、公民館の規模は大きくなるのではないか。

\### 検証

公民館の貸出可能部屋数と最寄り市営住宅との距離、人口割合、総人口の関係性を確認。

\### 結果

「貸出可能部屋数」と「最寄り市営住宅との距離」「人口割合」「総人口」のいずれとも相関関係はみられなかった。

\---

\## 分析途中で生まれた気づき

分析開始時は「地域活動の場」として公民館を扱っていた。

しかし、人口が多い地域でも公民館が配置されていない結果を受けて調査したところ、一部地域では活動センターやコミュニティセンターが同様の役割を担っていることが分かった。

\---

\## 考察

公民館がある地域に「最寄り市営住宅との距離」や「人口割合」「総人口」の関係性は観察できたが、「貸出可能部屋数」は公民館の規模を表す指標として収集したものの人口や市営住宅などの情報では説明できないことが分かった。

「貸出可能部屋数」は確かに公民館の規模を説明していると考える。規模には

* 地域の人口推移
* 地域の他行政施設の有無
* 公民館建設時期

などの他の要因が関係している可能性がある。

また、利用者状況も比較できれば施設規模に対して現在でも目的に合わせて積極的な利用がされているかの検証ができる可能性がある。

一覧形式のオープンデータには施設そのものの特徴情報が少ないため、公民館に対する分析が難しいことが確認できた。

\## 今後の課題

公民館に関するデータ収集の目的で集めていた元データに、公民館に関する記載がなかったり（無線LANは公民館にあるとHPに記載されていたが、無線LANのオープンデータには情報がなかった）、扱いきれなかった特徴量も分析中に存在した。

与えられたデータのみで分析を進めるならば、データを収集する際は中身も確認すること、扱いたい特徴量をリストアップし優先順位を決めて作業することが必要。
